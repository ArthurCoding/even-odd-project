Par ou Ímpar – Python

Projeto simples em Python que identifica se um número é par ou ímpar.

Objetivo

Receber um número.

Analisar o valor.

Informar se é par ou ímpar.

Como usar

Execute o arquivo Python:

python par_impar.py


Digite um número quando o programa pedir.

Lógica usada
Um número é par quando n % 2 == 0.

Caso contrário, é ímpar.



Exemplo

Entrada:

10


Saída:

10 é par.

Resumo
Projeto simples para treinar lógica básica em Python.
